# Project Charter

## Project Title: Fleet Vehicle GPS Tracking & Telematics System Implementation

- **Project ID:** PRJ-00451
- **Demand ID:** DEM-00173
- **Project Sponsor:** David Chen (Director, Fleet Management)
- **Project Manager:** TBD (Assigned by EPMO)

### 1. Project Overview & Goals
This project will procure and implement a telematics system for 250 fleet vehicles to provide real-time tracking, diagnostics, and driver behavior monitoring. The primary goals are to reduce operational costs, improve asset utilization, and enhance driver safety.

### 2. Scope
#### In Scope:
- Procurement and installation of telematics hardware.
- Configuration of the cloud software platform.
- Integration with the maintenance-logging system (one-way data flow).
- Training for all fleet managers and drivers.

#### Out of Scope:
- Integration with payroll or HR systems for time tracking.
- Customer-facing tracking features.
- Mobile device management for drivers.

### 3. Key Milestones & Timeline
- **Q1:** Vendor Selection & Procurement
- **Q2:** Hardware Installation & Platform Configuration
- **Q3:** Integration & User Acceptance Testing
- **Q4:** Training & Go-Live
- **Estimated Completion:** End of Fiscal Year

### 4. Budget
- **Approved Budget:** $225,000 (CapEx)
- **Contingency:** 10% ($22,500)

### 5. Project Team & Governance
- **Sponsor:** David Chen
- **Project Manager:** TBD
- **Steering Committee:** Technology Subcommittee
- **Key Stakeholders:** Head of Field Operations, IT Infrastructure Lead, Lead Mechanic

### 6. Success Criteria
- 10% reduction in fleet fuel consumption within 6 months post-implementation.
- All 250 vehicles are actively reporting data.
- System achieves >99% uptime.
